# Wemos-TIC

This is a custom board designed for the **Wemos D1 Mini (ESP8266)**. It serves as a multi-sensor telemetry node, featuring environment monitoring and energy tracking.

## Features
* **TIC Input:** Read "Télé-Information Client" data directly from Enedis **Linky** smart meters.
* **BME Sensor:** Onboard support for BME280/BME680 (Temperature, Humidity, Pressure).
* **External DHT22:** Dedicated header for an external DHT22 temperature and humidity sensor.
* **Form Factor:** Compact shield design for the Wemos D1 Mini ecosystem.
